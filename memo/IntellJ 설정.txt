스프링 부트 시작 전 설정
	햄버거 메뉴 - [Settings] 
		1. Auto import : 자동 import 설정 옵션 활성화 
			Add unambiguous imports on the fly	: 자동 import 기능
			Optimize imports on the fly			: 사용하지 않는 것은 자동으로 import 해제
			두 개의 항목을 체크

		2. Code Completion : 대.소문자 구분여부 체크
			Match case 체크 해제 
				사용하려는 클래스, 메서드 등을 대, 소문자를 구분없이 입력해도
				자동으로 알맞게 자동으로 고쳐줌.

		3. File encodings
			Global Encodings  
			Project Encodings 
			Default encoding for  properties files 
			세 개의 항목 모두 "UTF-8" 인코딩 방식으로 변경 

		4. Annotation Processors : 주석 처리기
			Enable annotation processing 체크
				외부로부터 받아오는 외부 라이브러리인 "Lombok"이 Anootation으로 작동하기 때문에
				"lombok" 라이브러리를 받아와서 제대로 사용하기 위해 설정해준다.

단축키
	Ctrl + D 			: 한 줄 복사
	Ctrl + X 			: 한 줄 삭제 및 잘라내기(복사 가능)
 	Shift + F6 		: 이름 변경 (변수, 메서드,  클래스 등등)
	Ctrl + Alt + L 		: 자동 줄 맞춤
	
	Ctrl + Shift + F 	: 검색		
	Ctrl + Shift + R 	: 바꾸기
	※ Shift 를 생략하면 해당 파일 안에서, Shift를 붙이면 해당 프로젝트 안에서 검색 및 바꾸기를 실행한다.
